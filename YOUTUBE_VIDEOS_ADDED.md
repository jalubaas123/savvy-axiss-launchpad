### Replacing Videos
If you want to add your own project demo videos:
1. Upload your demo video to YouTube
2. Get the video ID from the URL (e.g., `youtube.com/watch?v=VIDEO_ID`)
3. Update the videoUrl in `src/data/projectsData.ts`
4. Format: `https://www.youtube.com/embed/VIDEO_ID`

### Replacing Images
To use your own project screenshots:
1. Create `/public/projects/` directory
2. Add your images (recommended: 800x500px, PNG or JPG)
3. Update image paths in `src/data/projectsData.ts`
4. Use format: `/projects/your-image.png`
